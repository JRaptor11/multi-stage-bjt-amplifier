# Multi-Stage BJT Amplifier

This project focuses on the design and analysis of a multi-stage BJT amplifier to achieve high gain and proper impedance characteristics.

## Overview

The amplifier consists of multiple transistor stages designed to meet performance requirements for gain, input resistance, and output resistance.

## Key Features

- High voltage gain (>100)
- Multi-stage amplifier design
- Input/output impedance optimization
- Frequency response analysis

## Repository Structure

- `docs/` – design report and calculations  
- `images/` – circuit diagrams and plots  

## Skills Demonstrated

- Analog circuit design
- Transistor amplifier analysis
- Multi-stage system design
- Experimental validation
